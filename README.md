# pizza-sales-analysis-SQL-excel
Sales performance analysis of a fictional pizza store using SQL Server and Excel. Includes KPI evaluation, revenue trends, product performance insights, and business recommendations with an interactive dashboard.

Project Overview:

This project analyzes sales data from a fictional pizza store to evaluate business performance, identify revenue trends, and generate actionable insights.

The analysis was performed using:

>SQL Server for data querying and aggregation

>Microsoft Excel for data cleaning, KPI calculation, and dashboard creation

>PowerPoint for creating background of dashboard

Tools & Technologies:

>SQL Server

>Microsoft Excel

>Microsoft PowerPoint

Business Objectives:

>Analyze total revenue and overall order volume

>Identify top-performing pizza categories and products

>Evaluate peak sales hours and high-demand days

>Measure Average Order Value (AOV)

>Provide data-driven business recommendations

Key Insights:

>Weekends generate the highest revenue.

>Large-size pizzas contribute the maximum share of sales.

>Evening hours show peak order volume.

>Top 5 pizzas contribute to a majority of revenue (Pareto Principle).

Business Recommendations:

>Introduce combo offers during low-sales hours to increase order volume.

>Focus marketing efforts on high-margin pizza categories.

>Optimize inventory planning based on demand trends.
